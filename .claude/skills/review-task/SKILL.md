## Instructions
Read the plan I point you to. Do not modify code. Report on four things only,
with concrete recommendations:
1. Scope — correctly sized? Should it split into independent plans, or merge?
2. Subagents — each necessary? Could any run on a cheaper model?
3. Parallelism — which steps are independent and could run together?
4. Wasted context — what is loaded or researched that this task does not need?
